# obsidian_aipilot (Obsidian Plugin)

AI-powered inline completion plugin for Obsidian, adapted from Markpilot.

## Features

- Inline completion only (no chat view)
- Multi-provider support via AI SDK:
  - OpenAI
  - Anthropic
  - Google
  - Mistral
  - DeepSeek
  - Custom OpenAI-compatible endpoint
- Context-aware Markdown completion
- `Tab` to accept, `Escape` to reject

## Development

```bash
npm install
npm run dev
```

Build:

```bash
npm run build
```
